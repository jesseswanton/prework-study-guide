HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • <a href="url">link text

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd .. goes up one directory level
  • Download class materials (this pulls just new changes): go to bootcamp directory and then: git pull git@git.bootcampcontent.com:boot-camp-consortium-west-coast/FSF-PT-WEST-AUGUST-080524.git
  • Download class materials (this copies the whole thing): go to bootcamp directory and then: git clone git@git.bootcampcontent.com:boot-camp-consortium-west-coast/FSF-PT-WEST-AUGUST-080524.git
  • https://graphite.dev/guides/git-commit-to-new-branch
  • pull main, checkout/create your own branch & make changes, git add and commit your branch, pull main, checkout main, git merge, resolve conflicts, git add and commit, push your branch to origin main

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions can take input arguments and return output values. On the other hand, a method is a function that is associated with an object in object-oriented programming. Methods are functions that are called on objects and can modify or access the object's properties.
